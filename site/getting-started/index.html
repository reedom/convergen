    Getting Started¶

    Welcome to Convergen! This section will guide you through installation, your first conversion function, and basic usage patterns.

    What You'll Learn¶

    In this section, you'll discover:

    • Installation - Multiple ways to install and use Convergen
    • Quick Start - Your first conversion function in 5 minutes
    • Basic Examples - Common patterns and simple use cases

    Prerequisites¶

    Before getting started, make sure you have:

    • Go 1.21 or later installed
    • Basic familiarity with Go structs and interfaces
    • A Go project where you want to generate conversion functions

    Choose Your Path¶

    Start with the Installation guide to set up Convergen, then follow the Quick Start tutorial.

    Jump to Basic Examples to see Convergen in action with real code samples.

    Check out our Migration Guide for upgrade instructions and breaking changes.

    Next Steps¶

    Once you've completed the getting started section:

    1. Explore the User Guide for comprehensive annotation reference
    2. Check Real-World Examples for production patterns
    3. Review Best Practices for optimal usage
